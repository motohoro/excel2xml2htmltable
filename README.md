excel2xml2htmltable
=======================

（excelの表から作った）xmlをhtmlのテーブルにする手順

## Description
xsltを使ってxmlをhtmlのtableで表示する。

このやり方だと、excelの表に影響与えずhtmlのtableに表示する列や体裁を変えられる事が便利
